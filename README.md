## 📖 Descripción

**PANDA STATISTIC LIFE** es una plataforma desarrollada en **Python** utilizando **Streamlit**, diseñada para facilitar el análisis estadístico, bioestadístico, epidemiológico y demográfico de datos de salud.

El proyecto busca proporcionar una herramienta moderna, intuitiva y accesible para estudiantes, docentes, investigadores y profesionales de las ciencias de la salud, permitiendo realizar análisis de manera rápida y visual sin necesidad de programación avanzada.

Además, incorpora funcionalidades orientadas al procesamiento de datos provenientes del **Sistema Nacional de Información en Salud (SNIS) de Bolivia**, automatizando tareas de limpieza, organización y visualización.

---

# 🎯 Objetivos

* Facilitar el análisis estadístico aplicado a las ciencias de la salud.
* Apoyar la enseñanza de la bioestadística y epidemiología.
* Simplificar el procesamiento de reportes provenientes del SNIS.
* Proporcionar herramientas de análisis demográfico para la planificación sanitaria.
* Promover el uso de tecnologías abiertas en investigación y salud pública.

# ▶️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/omipaz1234-hue/PANDA-STATISTIC-LIFE.git
```

Ingresa al proyecto:

```bash
cd PANDA-STATISTIC-LIFE
```

Instala las dependencias:

```bash
pip install -r requirements.txt
```

Ejecuta la aplicación:

```bash
streamlit run app.py
```
