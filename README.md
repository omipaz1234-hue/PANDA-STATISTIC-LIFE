# 🐼 PANDA STATISTIC LIFE

**Estadística Descriptiva y Bioestadística para Salud Pública en Bolivia**

Basado en el libro *Estadística Descriptiva y de Salud* de Héctor Bayarre, Rben Hersford y Maritza Olivia.

### Características
- Análisis descriptivo completo
- Tablas de frecuencia
- Visualizaciones interactivas
- Canal Endémico con bandas
- Indicadores de salud
- Compatible con datos del SNIS (CSV/Excel)

### Cómo ejecutar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

Desarrollado para apoyar la vigilancia epidemiológica en Bolivia.
